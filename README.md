# QR Code Generator 🚀

A simple and efficient tool to generate QR codes from URLs or text. This project is designed to be lightweight and user-friendly.

## 🌟 Features
*   **Instant Generation:** Create QR codes in real-time.
*   **Custom Text/URL:** Supports any valid string or web address.
*   **Downloadable:** Save your generated QR codes as image files.
*   **Responsive Design:** Works on desktops, tablets, and mobile phones.

## 🛠️ Tech Stack
*   **HTML5 / CSS3** - Structure and styling.
*   **JavaScript** - Logic and interactivity.
*   **QR Library** - (e.g., QRCode.js or similar library you used).

## 🚀 How to Use
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/sauravsingh35/qr-code-generator.git](https://github.com/sauravsingh35/qr-code-generator.git)
    ```
2.  **Navigate to the folder:**
    ```bash
    cd qr-code-generator
    ```
3.  **Open the project:**
    Simply open `index.html` in your favorite web browser.
